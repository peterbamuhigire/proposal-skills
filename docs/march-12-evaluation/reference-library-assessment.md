# Cross-Cutting Reference Library Assessment

**Date:** 12 March 2026 (updated)

---

## Overview

The repository contains 41 substantive reference files across 11 reference areas, synthesising 67 sources (34 consulting/proposal books, 7 M&E sources, 2 impact evaluation books, 11 PPDA documents, and 13 AfDB web sources) into actionable frameworks. These files form the intellectual foundation of all proposal writing and are the single most valuable asset in the repository.
