# Sectors Routing

Use this directory to combine procurement mechanics with sector-domain context.

## Order of Operations
1. Identify the procurement framework first: `world-bank`, `afdb`, `ppda-uganda`, or `undp`.
2. Identify the sector next: `agriculture`, `education`, `energy`, `financial-services`, `governance`, `health`, `ict`, `transport-infrastructure`, or `water-sanitation`.
3. Load only the framework skill and the minimum set of sector skills relevant to the assignment.
4. Carry those assumptions into the numbered proposal section skills in the repo root.

## Expectations
- Framework skills control forms, thresholds, scoring, and compliance.
- Sector skills control terminology, stakeholder logic, methodology cues, and country-relevant framing.
- Use `uganda-country-profile.md` when Uganda-wide context materially strengthens the response.
